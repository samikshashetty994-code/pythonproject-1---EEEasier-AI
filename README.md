# ⚡ EEEasier AI

An AI-powered Electrical and Electronics Engineering assistant built using Python and Streamlit.

## 📖 Overview

EEEasier AI is designed to help Electrical and Electronics Engineering students learn concepts faster by providing AI-powered assistance for engineering questions, formulas, numerical problems, and notes.

## 🚀 Features

- 📷 Scan engineering questions using OCR
- 🤖 AI-powered question answering
- 📘 Formula Finder with Electrical and Electronics formulas
- 🧮 Numerical Solver with step-by-step solutions
- 📄 Upload and analyze PDF notes
- 🔌 Circuit Diagram Explainer
- 💬 AI Chat for EEE concepts

## 🛠 Technologies Used

- Python
- Streamlit
- OpenAI/OpenRouter API
- OCR (Pytesseract)
- PyMuPDF
- Pillow

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/EEEasier-AI.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your API key:

```
OPENROUTER_API_KEY=your_api_key_here
```

Run the application:

```bash
streamlit run main.py
```
